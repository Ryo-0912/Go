# 定義

```
const msg = "hello"
```

- 種類

  - 数値リテラル : 100, 1, 5_000(可読性のために、このような書き方もするが、解釈としては5000)
  - 文字列リテラル : "Go" 
  - ルーンリテラル : 'A', '世'


# まとめて定義

```
const (
	msg = "Hello"
	msgJP = "こんにちは"
	msgFR = "Bon Jour"
)
```

# 定数式

シフト演算

例

```
1 << 2
#=> 4
```

# 定数の型

| 種類 | 型なしの定数 | デフォルトの型 |
| - | - | - | 
| 複素数 | 1+4i, complex(4, 0) | complex128 |
| ルーン | 'A', '世' | rune |


# 右辺の省略 : グループ化された名前付き定数で用いられる

```
func main() {
	const (
		a = 1 + 2
		b
		c
	)
	fmt.Println(a, b, c)
}
```

省略された定数定義の右辺は、最後の省略されていない定数の右辺と同じになる
